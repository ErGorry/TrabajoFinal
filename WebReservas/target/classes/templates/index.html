Web de reserva e informacion de los viajes a Menorca

El objetivo de esta web es ofrecer una interfaz visual que conecte los tres microservicios del proyecto final del curso Master de Java de 160h.

Me he centrado especialmente en la funcionalidad REST dejando de lado la ambientacion y el front de este portal

El proyecto consiste de 3 microservicios que controlan tres entidades diferentes en BBDD aisladas unas de las otras. Lo que complica la transaccionalidad, esta ha sido mi manera de abordar el reto:

Tres Entidades con sus CRUD completos con metodos extra para que puedan tener interaccion entre ellos:

  • Hotel: int idHotel, String nombre, String categoria, double precio, boolean disponible
  • Vuelo: int idVuelo, String empresa, LocalDate fechaVuelo, Double precio, int plazasDisponibles
  • Reserva: int idReserva, String nombreCliente, String dni, int numeroPersonas, int idVuelo, int idHotel

Reserva es dependiente de las otras entidades y al crear una reservas.

Los microservicios se levantan en diferentes puertos y deben estar activos.
